Instale o PyInstaller
pip install pyinstaller`
Rode o seguinte comando para criar um executável(caso não queira que ele crie um terminal use a opção --noconsole)
pyinstaller --onefile nome-do-arquivo-python.py

Exemplo de comando para criar o arquivo na pasta "dist"
1) pip install pyinstaller 
2) pyinstaller --onefile -w             (Se precisar de interação de dela para execultar)
3) pyinstaller --onefile Gabarito.py            (SEM precisar de interação de dela para execultar)

Aqui estão alguns atalhos de teclado úteis do VS Code para Python:

Ctrl + Shift + P: Abre a palette de comandos.
Ctrl + Shift + N: Abre um novo arquivo.
Ctrl + Shift + T: Reabre o último arquivo fechado.
Ctrl + Shift + F: Pesquisa por texto em todos os arquivos do projeto.
Ctrl + /: Comenta/descomenta a linha atual.
Ctrl + Shift + L: Seleciona todas as ocorrências da seleção atual.
Shift + Alt + F: Formata o arquivo atual.
F5: Inicia o depurador.
F9: Adiciona ou remove um ponto de interrupção.
F12: Vai para a definição do símbolo atual.
Ctrl + espaço: Exibe a lista de preenchimento automático.
Ctrl + Shift + espaço: Exibe a lista de parâmetros de uma função.
Alt + Shift + F: Aplica autopep8 para formatar o código.
Ctrl + K + Z: Alterna para o modo de leitura (Zen).
Ctrl + K + S: Abre o menu para personalizar os atalhos de teclado.
